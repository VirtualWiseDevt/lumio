**client/src/types/auth.ts**:
```
export interface RegisterInput {
  name: string;
  email: string;
  phone: string;
  password: string;
  referralCode: string;
}

export interface LoginInput {
  email: string;
  password: string;
}

export interface AuthResponse {
  user: {
    id: string;
    name: string;
    email: string;
    phone?: string;
    referralCode?: string;
  };
  token: string;
}

export interface DeviceLimitResponse {
  deviceLimitReached: true;
  devices: Array<{
    id: string;
    deviceName: string;
    deviceType: string;
    lastActiveAt: string;
  }>;
}

export interface ReferralValidation {
  valid: boolean;
  referrerName?: string;
}
```

**client/src/api/auth.ts**:
1. `register(input: RegisterInput)`: POST /api/auth/register. Returns AuthResponse. Store token in localStorage("token").
2. `login(input: LoginInput)`: POST /api/auth/login. Returns AuthResponse | DeviceLimitResponse. If not deviceLimitReached, store token.
3. `validateReferralCode(code: string)`: GET /api/referrals/validate/{code}. Returns ReferralValidation. Catch 404 and return { valid: false }.
4. `forceLogin(input: LoginInput & { sessionId: string })`: POST /api/auth/force-login. For device limit override.

Import api from "./client".
