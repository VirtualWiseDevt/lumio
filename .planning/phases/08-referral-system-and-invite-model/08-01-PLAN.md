     id        String   @id @default(uuid())
     couponId  String
     userId    String
     paymentId String?
     createdAt DateTime @default(now())
     coupon  Coupon  @relation(fields: [couponId], references: [id])
     user    User    @relation(fields: [userId], references: [id], onDelete: Cascade)

**referral.service.ts** -- create with these exports:

1. `validateReferralCode(code: string)`: Look up User by referralCode. Return `{ valid: true, referrerName: "Kelvin M." }` or `{ valid: false }`. Format name as "FirstName LastInitial." (e.g., "Kelvin M."). If single-word name, return just the name.

2. `getReferralStats(userId: string)`: Query Referral table where referrerId = userId. Return `{ friendsJoined: number, creditsEarned: number }`. `friendsJoined` = count of referrals. `creditsEarned` = sum of creditAmount across all referrals.

3. `grantReferralCredit(tx: TxClient, refereeUserId: string)`: Called inside payment callback transaction. Find Referral where refereeId = refereeUserId AND isRedeemed = false. If found, get referrer's active subscription plan price. Calculate credit = Math.round(planPrice * 0.10). Update Referral: set creditAmount and isRedeemed = true. Increment referrer's referralCreditBalance by creditAmount. If referrer has no active subscription, use a default credit of 0 (skip granting). Use TxClient type pattern from subscription.service.ts: `type TxClient = Parameters<Parameters<PrismaClient["$transaction"]>[0]>[0]`.
