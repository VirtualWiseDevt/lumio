<task type="checkpoint:human-action" gate="blocking">
  <name>Task 2: Configure Cloudflare Referer check</name>
  <action>
Configure Cloudflare to restrict access to the R2 video CDN domain to requests with a Referer from lumio.tv only (LOCKED DECISION).

This prevents hotlinking -- other websites cannot embed or link to video segments.
  </action>
  <instructions>
1. Log in to Cloudflare Dashboard
2. Select the domain used for R2 video CDN (the custom domain serving R2 bucket content)
3. Navigate to Security -> WAF -> Custom Rules
4. Create a new rule:
   - Rule name: "Block non-lumio.tv referers"
   - Expression: (http.referer ne "" and not http.referer contains "lumio.tv")
   - Action: Block
   - OR alternatively, use a Referer check that allows empty referer (for direct player requests) but blocks foreign referers:
     - Expression: (http.referer ne "" and not http.referer contains "lumio.tv" and not http.referer contains "localhost")
     - Include localhost for development
5. Deploy the rule
6. Verify by testing: curl -H "Referer: https://evil.com" [your-cdn-url]/some-path should be blocked
  </instructions>
  <resume-signal>Type "configured" once the Cloudflare Referer rule is deployed, or "skip" to handle later</resume-signal>
