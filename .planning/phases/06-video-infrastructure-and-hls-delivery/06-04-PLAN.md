  <action>
1. Create api/src/services/stream.service.ts:

   getStreamPlaylist(contentId: string, episodeId?: string): Promise<string>
   - Determine HLS prefix: episodes use `videos/${contentId}/episodes/${episodeId}/hls/`, content uses `videos/${contentId}/hls/`
   - Download master.m3u8 from R2 (use GetObjectCommand, read Body to string)
   - Parse the master playlist line by line
   - For each line that references a quality playlist (e.g., "360p/playlist.m3u8"):
     - Replace with API URL: `/api/stream/${contentId}/360p${episodeId ? '?episode=' + episodeId : ''}`
   - Return the rewritten master playlist content

   getQualityPlaylist(contentId: string, quality: string, episodeId?: string): Promise<string>
   - Determine HLS prefix based on contentId/episodeId
   - Download quality playlist from R2: `${prefix}${quality}/playlist.m3u8`
   - Parse line by line
   - For each line that references a .ts segment (e.g., "segment001.ts"):
     - Generate presigned URL for: `${prefix}${quality}/segment001.ts`
     - Replace the segment filename with the full presigned URL
   - Return the rewritten playlist content

   Import { generatePresignedDownloadUrl } from "./r2.service.js"
   Import { GetObjectCommand } from "@aws-sdk/client-s3"
   Import { r2Client, R2_BUCKET_NAME } from "../config/r2.js"

2. Create api/src/routes/stream.routes.ts:

   GET /api/stream/:contentId
   - Protected by requireAuth (regular users, not admin-only)
   - Query params: ?episode=episodeId (optional)
   - Look up Content by id, verify it has hlsKey (or episode has hlsKey if episodeId)
   - If no hlsKey, return 404 "Content not yet transcoded"
   - Call getStreamPlaylist(contentId, episodeId)
   - Set Content-Type: application/vnd.apple.mpegurl
   - Set Cache-Control: public, max-age=60 (LOCKED DECISION: .m3u8 playlists use 60s TTL)
   - Return playlist text as response body

   GET /api/stream/:contentId/:quality
   - Protected by requireAuth
   - Params: contentId, quality (e.g., "360p")
   - Query params: ?episode=episodeId (optional)
   - Call getQualityPlaylist(contentId, quality, episodeId)
   - Set Content-Type: application/vnd.apple.mpegurl
   - Set Cache-Control: public, max-age=60 (LOCKED DECISION: .m3u8 playlists use 60s TTL)
   - Return playlist text

   Use Express 5 Router. Import requireAuth from middleware.

3. Register in api/src/routes/index.ts:
   - Import { streamRouter } from "./stream.routes.js"
   - Add: app.use("/api/stream", streamRouter)
