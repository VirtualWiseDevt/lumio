Create api/src/services/r2.service.ts with these functions:

1. generatePresignedUploadUrl(key: string, contentType: string, expiresIn = 3600): Promise<string>
   - Use @aws-sdk/s3-request-presigner getSignedUrl with PutObjectCommand
   - Set ContentType on the command
   - Returns the presigned URL string

2. generatePresignedDownloadUrl(key: string, expiresIn = 14400): Promise<string>
   - 14400 = 4 hours default TTL (per context decisions)
   - Use GetObjectCommand with getSignedUrl

3. uploadToR2(key: string, body: Buffer | Readable, contentType: string): Promise<void>
   - Use PutObjectCommand directly (for uploading transcoded segments from server)

4. deleteFromR2(key: string): Promise<void>
   - Use DeleteObjectCommand

5. deleteR2Prefix(prefix: string): Promise<void>
   - Use ListObjectsV2Command to list all objects with prefix
   - Delete each with DeleteObjectCommand
   - Handle pagination (IsTruncated + ContinuationToken)

6. listR2Objects(prefix: string): Promise<string[]>
   - Use ListObjectsV2Command, return array of keys
   - Handle pagination

7. headR2Object(key: string): Promise<{ contentLength: number; contentType: string } | null>
   - Use HeadObjectCommand, return metadata or null if not found
   - Catch NoSuchKey error and return null

All functions import r2Client and R2_BUCKET_NAME from ../config/r2.js.
Import commands from @aws-sdk/client-s3 and getSignedUrl from @aws-sdk/s3-request-presigner.
Use ESM imports with .js extensions.

  <files>api/src/routes/video-upload.routes.ts, api/src/routes/index.ts</files>
  <action>
Create api/src/routes/video-upload.routes.ts:

1. POST /api/admin/video-upload/presign
   - Protected by requireAuth + requireAdmin
   - Body: { contentId: string, episodeId?: string, filename: string, contentType: string, fileSize: number }
   - Validate contentType is video/mp4, video/x-matroska, or video/quicktime
   - Validate fileSize <= 5 * 1024 * 1024 * 1024 (5GB)
   - Verify content exists (and episode exists if episodeId provided)
   - Generate R2 key: `videos/${contentId}/raw/${episodeId ? episodeId + '/' : ''}source${ext}` where ext is derived from filename
   - Actually simplify: key = `videos/${contentId}${episodeId ? '/episodes/' + episodeId : ''}/raw/source.${ext}`
     where ext = filename.split('.').pop()
   - Call generatePresignedUploadUrl(key, contentType)
   - Return { uploadUrl: string, key: string }

2. POST /api/admin/video-upload/confirm
   - Protected by requireAuth + requireAdmin
   - Body: { contentId: string, episodeId?: string, key: string }
   - Verify the file actually exists in R2 via headR2Object(key)
   - If not found, return 400 "File not found in storage"
   - Run ffprobe validation on the file:
     - Download a small portion is not practical for large files
     - Instead, just verify the file exists via HEAD and trust the content type
     - ffprobe validation happens at transcode time (when file is downloaded)
   - Update database:
     - If episodeId: update Episode set sourceVideoKey = key
     - Else: update Content set sourceVideoKey = key
   - Return { success: true, key: string }

Register in api/src/routes/index.ts:
- Import videoUploadRouter
- app.use("/api/admin/video-upload", videoUploadRouter)

Use Zod schemas for request validation inline in the route file.
Use Express 5 Router (import { Router } from "express").
Apply requireAuth and requireAdmin middleware via router.use().
