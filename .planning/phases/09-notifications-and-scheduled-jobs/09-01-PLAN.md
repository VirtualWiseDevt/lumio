<task type="auto">
  <name>Task 2: Email transport factory and email service with templates</name>
  <files>api/src/config/email.ts, api/src/services/email.service.ts</files>
  <action>
1. Create `api/src/config/email.ts` -- dual-mode transport factory mirroring the mpesa.ts pattern:

   - Define `EmailMessage` interface: `{ to: string; subject: string; html: string; text?: string }`
   - Define `EmailTransport` interface: `{ send(message: EmailMessage): Promise<void> }`
   - `SmtpTransport` class: uses `nodemailer.createTransport()` with env.SMTP_HOST, env.SMTP_PORT, env.SMTP_USER, env.SMTP_PASS. Sends from `"Lumio" <${env.SMTP_FROM}>`.
   - `ConsoleTransport` class: logs `[EMAIL] To:`, `[EMAIL] Subject:`, `[EMAIL] Body preview:` (first 200 chars of html), `[EMAIL] ---` to console.
   - `getEmailTransport()` factory: singleton pattern, returns SmtpTransport if `env.EMAIL_TRANSPORT === "smtp"`, else ConsoleTransport.
   - Import nodemailer as `import nodemailer from "nodemailer"` (default import, esModuleInterop is true). If that causes issues at runtime, use `import * as nodemailer from "nodemailer"`.
   - Import type Transporter from nodemailer for SmtpTransport internal field.

2. Install nodemailer and types:
   ```bash
   cd api && npm install nodemailer && npm install -D @types/nodemailer
   ```

3. Create `api/src/services/email.service.ts` -- shared email utility:

   - Import `getEmailTransport` from `../config/email.js`
   - Create a `sendEmail(to: string, subject: string, html: string, text?: string)` wrapper that calls `getEmailTransport().send(...)`. This is the single entry point.
   - Create template builder functions, each returning `{ subject: string; html: string; text: string }`:

   a) `buildWelcomeEmail(name: string)` -- NOTF-01
      Subject: "Welcome to Lumio!"
      Content: Greeting, welcome message, encourage browsing content, link to app

   b) `buildPaymentSuccessEmail(name: string, data: { amount: number; planName: string; duration: string; expiresAt: Date; mpesaReceipt: string | null; couponDiscount?: number; creditsUsed?: number })` -- NOTF-02
      Subject: "Payment Confirmed - Lumio"
      Content: Full breakdown with amount, plan name, duration, new expiry date, M-Pesa receipt number. Include coupon/credit deduction lines if applicable.

   c) `buildPaymentFailureEmail(name: string, data: { planName: string; amount: number })` -- NOTF-03
      Subject: "Payment Failed - Lumio"
      Content: Payment failed notice, retry suggestion ("try again on the billing page"), support contact (support@lumio.tv)

   d) `buildPreExpiryEmail(name: string, data: { planName: string; expiresAt: Date; daysRemaining: number })` -- NOTF-04
      Subject: "Your Lumio subscription expires soon"
      Content: Warn about upcoming expiry, show date, link to /billing to renew

   e) `buildPostExpiryEmail(name: string, data: { planName: string; expiredAt: Date })` -- NOTF-05
      Subject: "Your Lumio subscription has expired"
      Content: Subscription expired notice, reactivation link to /billing, mention what they're missing

   f) `buildReferralRewardEmail(name: string, data: { refereeName: string; creditsEarned: number; newBalance: number })` -- NOTF-06
      Subject: "You earned referral credits - Lumio"
      Content: Congratulate, show who referred, credits earned, new balance

   g) `buildPasswordResetEmail(name: string, data: { resetUrl: string })` -- bonus (wires existing forgotPassword TODO)
      Subject: "Reset your Lumio password"
      Content: Reset link, 1-hour expiry warning, ignore-if-not-you note

   **HTML template structure for all emails:**
   - Wrapper div: `font-family: Arial, sans-serif; max-width: 600px; margin: 0 auto; padding: 20px;`
   - Header: `<h1 style="color: #e50914; font-size: 28px; margin-bottom: 0;">Lumio</h1>` (Netflix-red brand color)
   - Horizontal rule separator
   - Body content with paragraphs
   - CTA button where relevant: `display: inline-block; background: #e50914; color: white; padding: 12px 24px; text-decoration: none; border-radius: 4px;`
   - Footer: `<hr /><p style="color: #999; font-size: 12px;">This is a transactional email from Lumio. You received this because you have an account at lumio.tv.</p>`
   - Each template also returns a `text` field (plain text version, no HTML)

   **IMPORTANT patterns:**
   - Use .js extensions on all imports (ESM)
   - Do NOT use template engines -- pure template literals
   - Format dates using `toLocaleDateString('en-KE', { year: 'numeric', month: 'long', day: 'numeric' })`
   - Format amounts as `KES ${amount.toLocaleString()}`
