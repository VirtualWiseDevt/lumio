<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
Complete Netflix-style client browsing experience with:
- Home page: hero banner with auto-rotating featured content + content rows
- Movies, Series, Documentaries pages: type-specific hero + filtered rows
- Content cards with 500ms hover popover showing metadata and action buttons
- Detail modal (intercepting routes) with full content info, episode list for series, "More Like This"
- Full-screen search overlay (Ctrl+K) with real-time grouped results
- Live TV page with channel grid and pulsing LIVE badges
- Dark theme throughout
  </what-built>
  <how-to-verify>
1. Start both servers: `npm run dev` in api/ and `npm run dev` in client/
2. Visit http://localhost:3000

**Home Page (BRWS-01, BRWS-02):**
- Hero banner shows featured content with backdrop image
- Auto-rotates every ~8 seconds (watch for dot indicator change)
- If content has trailerUrl, video should crossfade in after ~1.5s (muted)
- Play and More Info buttons visible
- Content rows scroll horizontally with arrow buttons on hover
- Cards snap to boundaries (no half-cards visible)

**Content Type Pages (BRWS-03, BRWS-04, BRWS-05):**
- Click Movies, Series, Documentaries in navbar
- Each page has its own hero banner with type-specific content
- Rows show only content of that type

**Hover Popover (BRWS-06):**
- Hover over a content card for ~500ms
- Popover appears with enlarged card, backdrop image, action buttons, metadata
- Move mouse to popover -- it stays open
- Move mouse away -- it closes

**Detail Modal (BRWS-07, BRWS-08, BRWS-09):**
- Click a content card -- modal overlay appears, URL changes to /title/{id}
- Modal shows: title, year, duration, quality, age rating, description, cast, genres
- If it's a series: season dropdown + episode list visible
- "More Like This" grid shows 6 similar titles
- Click X or backdrop to close -- URL returns to previous page
- Copy /title/{id} URL and open in new tab -- full page renders (not modal)

**Search (BRWS-10):**
- Press Ctrl+K -- search overlay opens
- Type a query -- results appear grouped by type after brief delay
- Click a result -- navigates to detail
- Press Escape -- overlay closes

**Live TV (BRWS-11, BRWS-12):**
- Click Live TV in navbar
- Channel cards in grid, organized by category headers
- Each card has channel image, name, and pulsing red LIVE badge

**General:**
- Dark theme consistent across all pages
- No console errors
- Responsive: resize browser to test mobile breakpoints
