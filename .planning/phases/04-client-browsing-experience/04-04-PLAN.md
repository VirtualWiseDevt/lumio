interface UseContentRowReturn {

**1. `client/src/hooks/use-hover-popover.ts`:**
```typescript
"use client";

interface UseHoverPopoverReturn {
  activeId: string | null;
  cardRect: DOMRect | null;
  onMouseEnter: (id: string, element: HTMLElement) => void;
  onMouseLeave: () => void;
  onPopoverMouseEnter: () => void;
  onPopoverMouseLeave: () => void;
}
```

Implementation:
- 500ms delay before setting `activeId` (setTimeout)
- Stores the card's `getBoundingClientRect()` when hover starts (for popover positioning)
- `onMouseLeave` from card: sets a 100ms closing delay (not instant -- gives time to move to popover)
- `onPopoverMouseEnter`: cancels the closing delay (popover stays open)
- `onPopoverMouseLeave`: closes immediately (set activeId to null)
- Cleanup: clear all timeouts on unmount
- This pattern creates a shared hover zone: card -> small gap -> popover, mouse can traverse without flicker
