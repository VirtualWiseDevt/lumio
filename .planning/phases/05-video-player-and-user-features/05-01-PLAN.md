3. Create `api/src/services/progress.service.ts`:
   - `saveProgress(userId, data)`: Upsert WatchProgress using @@unique([userId, contentId, episodeId]). Set completed = true if timestamp/duration >= 0.9. Accept: contentId (string), episodeId (string | null), timestamp (number), duration (number).
   - `getProgress(userId, contentId, episodeId?)`: Find unique progress record. Return { timestamp, duration, completed } or null.
   - `getContinueWatching(userId, months?)`: Query WatchProgress where completed=false, apply hybrid threshold filter: only include if `timestamp >= max(120, duration * 0.05)` AND `duration >= 120` (content >= 2 min). Filter by updatedAt within N months (default 3, accept 3/6/12). Order by updatedAt DESC. Join with Content to return content metadata. For series, also resolve the episode title/thumbnail via Episode join.
   - Import PrismaClient from `../../generated/prisma/client.js`

4. Create `api/src/validators/progress.validators.ts`:
   - `saveProgressSchema`: z.object({ contentId: z.string().uuid(), episodeId: z.string().uuid().nullable().optional(), timestamp: z.number().int().min(0), duration: z.number().int().min(0) })
   - `continueWatchingQuerySchema`: z.object({ months: z.coerce.number().int().refine(v => [3, 6, 12].includes(v)).optional().default(3) })
  </action>
