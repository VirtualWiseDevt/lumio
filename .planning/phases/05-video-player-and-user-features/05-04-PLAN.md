1. Create `client/src/hooks/use-hls.ts`:
   - MUST dynamically import hls.js inside useEffect to avoid SSR "self is not defined" error
   - Accept videoRef (RefObject<HTMLVideoElement | null>) and src (string | null)
   - If src ends with ".m3u8": dynamically `import("hls.js")`, check Hls.isSupported(), create instance with maxBufferLength: 30, maxMaxBufferLength: 60, loadSource, attachMedia
   - On MANIFEST_PARSED: video.play().catch(() => {}) for autoplay
   - On ERROR: handle fatal errors (NETWORK -> startLoad, MEDIA -> recoverMediaError, default -> destroy)
   - Safari fallback: if !Hls.isSupported() but canPlayType("application/vnd.apple.mpegurl"), set video.src directly
   - If src is NOT .m3u8: set video.src directly (MP4)
   - Store hls instance in useRef, destroy on cleanup
   - Return hlsRef for external access if needed

2. Create `client/src/hooks/use-video-player.ts`:
   - Accept videoRef
   - Wire native video element events to Zustand player store:
     - play -> setPlaying(true), pause -> setPlaying(false)
     - timeupdate -> setCurrentTime(video.currentTime)
     - durationchange -> setDuration(video.duration)
     - waiting -> setBuffering(true) + setBufferingStartTime(Date.now())
     - playing -> setBuffering(false) + setBufferingStartTime(null)
     - progress -> calculate buffered end and setBuffered(end/duration)
   - Cleanup all listeners on unmount
   - Call store.reset() on unmount

3. Create `client/src/components/player/VideoPlayer.tsx`:
   - "use client" directive
   - Renders fixed inset-0 z-50 bg-black container (overlay approach per RESEARCH.md)
   - Contains <video> element with ref, playsInline, autoPlay
   - Uses useHls hook with video source URL
   - Uses useVideoPlayer hook for event wiring
   - Renders PlayerControls as overlay child
   - Renders BufferingIndicator when isBuffering
   - Props: src (string), title (string), contentId (string), episodeId (string | null), onClose (() => void), onNextEpisode? (() => void)
   - Mobile touch zones: three invisible div overlays (left/center/right thirds) with onClick handlers for rewind/play-pause/skip

4. Create `client/src/app/watch/[id]/page.tsx`:
   - "use client" directive
   - Extract contentId from params, episodeId from searchParams
   - Fetch content detail using fetchTitleDetail(contentId) from api/content.ts
   - Fetch saved progress using getProgress(contentId, episodeId) from api/progress.ts
   - Determine video source: if episodeId, find episode.videoUrl from content.seasons; else content.streamUrl or first available videoUrl
   - Set initial currentTime from saved progress (resume playback)
   - Render VideoPlayer with determined source
   - onClose: router.back()
   - Handle loading state with black screen + centered spinner
  </action>
