  <action>
    1. Create admin/src/api/upload.ts:
       - `uploadPoster(file: File)`: POST /api/admin/upload/poster as FormData with field 'image'. Returns { paths: { original, large, medium, thumbnail } }.
       - `uploadBackdrop(file: File)`: POST /api/admin/upload/backdrop as FormData with field 'image'. Returns { paths: { original, large, medium } }.
       - Use apiClient with Content-Type multipart/form-data (let axios set the boundary automatically -- do NOT manually set Content-Type header)

    2. Create admin/src/components/shared/ImageUploader.tsx:
       - Props: type ('poster' | 'backdrop'), value (current image path or null), onChange (callback with paths object), label (string)
       - Uses react-dropzone for drag-and-drop zone
       - Accept: image/jpeg, image/png, image/webp
       - maxSize: 10MB
       - States: idle (dropzone), uploading (spinner), uploaded (preview + replace button)
       - On file drop: call uploadPoster or uploadBackdrop based on type, show upload progress
       - After upload: show thumbnail preview of the uploaded image (use /api/media/{paths.thumbnail || paths.medium} as src)
       - "Replace" button clears the preview and shows dropzone again
       - Error handling: show toast on upload failure, show inline error for rejected file types
       - Poster dropzone: vertical aspect ratio hint (2:3)
       - Backdrop dropzone: horizontal aspect ratio hint (16:9)
       - Install react-dropzone if not already: `npm install react-dropzone --workspace=admin`
  </action>
  <verify>
    `cd admin && npx tsc --noEmit` compiles clean.
  </verify>
  <done>ImageUploader component handles drag-and-drop upload for both poster and backdrop types with preview, replace, and error handling.</done>

       - Uses React Hook Form with zodResolver
       - Zod schema matching API's createContentSchema:

         - description (optional, textarea)
         - releaseYear (optional, number 1900-2100)
         - duration (optional, number, in minutes)
         - ageRating (optional, select: G, PG, PG-13, R, NC-17, or custom input)
         - quality (optional, select: SD, HD, FHD, 4K)
         - categories (multi-select from categories API -- use combobox or multi-select dropdown)
         - cast (tag-style input: type name, press Enter to add, click X to remove)
         - director (optional, text input)
         - trailerUrl (optional, URL input)
         - videoUrl (optional, text input -- "Video file reference, set in Phase 6")
         - Poster image: ImageUploader type="poster"
         - Backdrop image: ImageUploader type="backdrop"
         - isPublished toggle (shadcn Switch)
       - On submit: calls createContent or updateContent, shows success toast, calls onSuccess
       - Categories dropdown: fetch from listCategories, allow selecting multiple
       - Form layout: two-column on desktop (left: main fields, right: images + metadata), single column on mobile
       - "Save as Draft" and "Save & Publish" buttons at bottom
