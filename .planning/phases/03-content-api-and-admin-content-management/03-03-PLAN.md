  <name>Task 1: Content validators, service, and routes</name>
  <files>
    api/src/validators/content.validators.ts
    api/src/services/content.service.ts

  <action>
    1. Create api/src/validators/content.validators.ts:
       - `contentQuerySchema`: z.object for GET /content query params:
         - type: z.nativeEnum(ContentType) -- required (each page filters by type)
         - status: z.enum(['all', 'published', 'draft']).default('all')
         - category: z.string().optional()
         - search: z.string().optional()
         - page: z.coerce.number().min(1).default(1)
         - limit: z.coerce.number().min(1).max(100).default(20)
         - sortBy: z.enum(['title', 'releaseYear', 'quality', 'createdAt']).default('createdAt')
         - sortOrder: z.enum(['asc', 'desc']).default('desc')
       - `createContentSchema`: z.object for POST /content:
         - type: z.nativeEnum(ContentType)
         - title: z.string().min(1).max(255).trim()
         - description: z.string().optional()
         - releaseYear: z.coerce.number().int().min(1900).max(2100).optional()
         - duration: z.coerce.number().int().positive().optional()
         - ageRating: z.string().max(10).optional()
         - quality: z.string().optional() (accept SD, HD, FHD, 4K)
         - categories: z.array(z.string()).default([])
         - cast: z.array(z.string()).default([])
         - director: z.string().optional()
         - trailerUrl: z.string().url().optional().or(z.literal(''))
         - videoUrl: z.string().optional()
         - streamUrl: z.string().optional() (for channels)
         - posterPortrait: z.string().optional() (image path from upload)
         - posterLandscape: z.string().optional() (image path from upload)
         - isPublished: z.boolean().default(false)
       - `updateContentSchema`: createContentSchema.partial().omit({ type: true }) -- can't change type after creation
       - Import ContentType from generated Prisma client

    2. Create api/src/services/content.service.ts:
       - `listContent(query)`: Prisma findMany with:
         - where: { type, isPublished (mapped from status), categories has (if category provided), title contains (if search, case-insensitive) }
         - orderBy: { [sortBy]: sortOrder }
         - skip/take for pagination
         - Also count total for pagination metadata
         - Return { data, total, page, totalPages }
         - For SERIES type, include `_count: { select: { seasons: true } }` so admin sees season count
       - `getContent(id)`: findUnique with seasons (include count) for series
       - `createContent(data)`: prisma.content.create
       - `updateContent(id, data)`: prisma.content.update
       - `deleteContent(id)`: prisma.content.delete (cascades to seasons/episodes)
       - `publishContent(id)`: update isPublished to true
       - `unpublishContent(id)`: update isPublished to false

    3. Create api/src/routes/content.routes.ts:
       - Apply `requireAuth, requireAdmin` to all routes via router.use()
       - GET / -- parse query with contentQuerySchema, call listContent
       - GET /:id -- call getContent, return 404 if not found
       - POST / -- parse body with createContentSchema, call createContent, return 201
       - PUT /:id -- parse body with updateContentSchema, call updateContent
       - DELETE /:id -- call deleteContent, return 204
       - PATCH /:id/publish -- call publishContent
       - PATCH /:id/unpublish -- call unpublishContent
       - Use inline try/catch pattern per handler (same as auth routes)

    4. Update api/src/routes/index.ts:
       - Import contentRouter, categoryRouter
       - Register: `app.use("/api/admin/content", contentRouter)`
       - Register: `app.use("/api/admin/categories", categoryRouter)` (categoryRouter created in Task 2)

    ESM RULES: .js extensions on all imports. Import Prisma types from generated path.
  </action>
  <verify>

    - `curl -X POST -H "Authorization: Bearer {admin_token}" -H "Content-Type: application/json" -d '{"type":"MOVIE","title":"Test Movie","releaseYear":2024}' http://localhost:5000/api/admin/content` -- returns 201 with created movie
  </verify>
