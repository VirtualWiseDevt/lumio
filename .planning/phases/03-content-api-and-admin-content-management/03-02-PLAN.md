  <action>
    1. Create admin/src/api/client.ts:
       - Axios instance with baseURL "/api" (proxied by Vite to localhost:5000)
       - Request interceptor: attach `Authorization: Bearer {token}` from localStorage
       - Response interceptor: on 401, clear localStorage token and redirect to /login
       - Export as `apiClient`

    2. Create admin/src/api/auth.ts:
       - `login(email: string, password: string)`: POST /api/admin/login, returns { user, token }
       - `getProfile()`: GET /api/sessions (to verify token is still valid -- reuse existing endpoint)

    3. Create admin/src/hooks/useAuth.tsx:
       - React context with AuthProvider component
       - State: { user, token, isAuthenticated, isLoading }
       - On mount: check localStorage for token, validate with getProfile(), set user or clear token
       - `login(email, password)`: calls auth API, stores token in localStorage, sets user state
       - `logout()`: clears localStorage, sets user to null
       - Export AuthProvider and useAuth hook

    4. Create route files using TanStack Router file-based routing:

       admin/src/routes/__root.tsx:
       - Root layout that renders `<Outlet />` with Toaster from sonner
       - Wraps children in QueryClientProvider

       admin/src/routes/login.tsx:
       - Login form with email + password inputs (shadcn Input, Button, Card)
       - Calls useAuth().login on submit
       - Shows error toast on failure via sonner
       - Redirects to / on success
       - If already authenticated, redirect to /

       admin/src/routes/_authenticated.tsx:
       - Layout route that checks useAuth().isAuthenticated
       - If not authenticated, redirect to /login using TanStack Router's redirect
       - Renders sidebar + header + main content area (Outlet)
       - Layout: fixed sidebar (240px) on left, header on top, scrollable content area

       admin/src/routes/_authenticated/index.tsx:
       - Dashboard placeholder page: "Welcome to Lumio Admin" with basic stats cards (placeholder data)

    5. Create layout components:

       Sidebar.tsx:
       - Fixed left sidebar, 240px wide, dark background (slate-900)
       - Lumio logo/text at top
       - Navigation links using lucide-react icons:
         - Dashboard (LayoutDashboard icon)
         - Movies (Film icon)
         - Series (Tv icon)
         - Documentaries (BookOpen icon)
         - TV Channels (Radio icon)
         - Settings (Settings icon) -- with sub-item "Categories"
       - Active link highlighted
       - Logout button at bottom
       - Links point to /movies, /series, /documentaries, /channels, /settings/categories
       - These routes don't exist yet -- they'll be created in later plans. For now the links are there.

       Header.tsx:
       - Top bar showing current page title (from route meta or breadcrumbs)
       - Admin user name/email on the right
       - Simple, minimal

       PageContainer.tsx:
       - Wrapper with consistent padding, max-width, page title prop
       - Used by all content pages for consistency

    6. Generate route tree:
       - Run TanStack Router's code generation or manually create routeTree.gen.ts
       - The route tree connects all route files to the router

    IMPORTANT: TanStack Router v1 uses file-based routing. Check the installed version and use the correct API. If using @tanstack/react-router, the createFileRoute or createLazyRoute pattern is standard. Install @tanstack/router-vite-plugin for automatic route tree generation if available, or use @tanstack/router-plugin/vite.
